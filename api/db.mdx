The `Db` struct represents a SlateDB database instance and provides methods for reading and writing data.

## Opening a Database

### open

```rust
pub async fn open<P: Into<Path>>(
    path: P,
    object_store: Arc<dyn ObjectStore>,
) -> Result<Self, Error>
```

Open a new database with default options.

<ParamField path="path" type="P: Into<Path>" required>
  The path to the database in object storage
</ParamField>

<ParamField path="object_store" type="Arc<dyn ObjectStore>" required>
  The object store implementation to use for persistence
</ParamField>

<ResponseField name="Result" type="Result<Db, Error>">
  Returns a `Db` instance on success, or an error if the database could not be opened
</ResponseField>

```rust
use slatedb::{Db, Error};
use slatedb::object_store::memory::InMemory;
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<(), Error> {
    let object_store = Arc::new(InMemory::new());
    let db = Db::open("test_db", object_store).await?;
    Ok(())
}
```

### builder

```rust
pub fn builder<P: Into<Path>>(
    path: P,
    object_store: Arc<dyn ObjectStore>,
) -> DbBuilder<P>
```

Creates a new builder for configuring and opening a database.

<ParamField path="path" type="P: Into<Path>" required>
  The path to the database
</ParamField>

<ParamField path="object_store" type="Arc<dyn ObjectStore>" required>
  The object store to use for the database
</ParamField>

<ResponseField name="DbBuilder" type="DbBuilder<P>">
  Returns a builder that can be configured before opening the database
</ResponseField>

```rust
use slatedb::{Db, Error};
use slatedb::object_store::memory::InMemory;
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<(), Error> {
    let object_store = Arc::new(InMemory::new());
    let db = Db::builder("/tmp/test_db", object_store)
        .build()
        .await?;
    Ok(())
}
```

## Scan Operations

### scan

```rust
pub async fn scan<K, T>(&self, range: T) -> Result<DbIterator, Error>
where
    K: AsRef<[u8]> + Send,
    T: RangeBounds<K> + Send,
```

Scan a range of keys using default scan options.

<ParamField path="range" type="T: RangeBounds<K>" required>
  The range of keys to scan (e.g., `"a".."z"` or `..`)
</ParamField>

<ResponseField name="Result" type="Result<DbIterator, Error>">
  Returns a `DbIterator` for iterating over the key-value pairs in the range
</ResponseField>

```rust
db.put(b"a", b"a_value").await?;
db.put(b"b", b"b_value").await?;

let mut iter = db.scan("a".."b").await?;
let kv = iter.next().await?.unwrap();
assert_eq!(kv.key.as_ref(), b"a");
assert_eq!(kv.value.as_ref(), b"a_value");
```

## Snapshot and Transaction Operations

### snapshot

```rust
pub async fn snapshot(&self) -> Result<Arc<DbSnapshot>, Error>
```

Create a read-only snapshot of the database.

<ResponseField name="Result" type="Result<Arc<DbSnapshot>, Error>">
  Returns a `DbSnapshot` representing a consistent view of the database
</ResponseField>

```rust
db.put(b"key1", b"value1").await?;
let snapshot = db.snapshot().await?;

// Snapshot provides read-only access
let value = snapshot.get(b"key1").await?;
assert_eq!(value, Some(Bytes::from("value1")));

// Further writes don't affect the snapshot
db.put(b"key2", b"value2").await?;
assert_eq!(snapshot.get(b"key2").await?, None);
```

### begin

```rust
pub async fn begin(&self, isolation_level: IsolationLevel) -> Result<DbTransaction, Error>
```

Begin a new transaction with the specified isolation level.

<ParamField path="isolation_level" type="IsolationLevel" required>
  The isolation level (`Snapshot` or `SerializableSnapshot`)
</ParamField>

<ResponseField name="Result" type="Result<DbTransaction, Error>">
  Returns a `DbTransaction` for performing transactional operations
</ResponseField>

```rust
use slatedb::IsolationLevel;

let txn = db.begin(IsolationLevel::Snapshot).await?;
txn.put(b"key", b"value")?;
txn.commit().await?;
```
