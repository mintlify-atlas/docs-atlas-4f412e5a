---
title: 'DbBuilder'
description: 'Builder for configuring and opening a SlateDB database'
---

The `DbBuilder` struct provides a fluent API for configuring database options before opening a database instance.

## Creating a Builder

### new

```rust
pub fn new(path: P, object_store: Arc<dyn ObjectStore>) -> Self
```

Creates a new `DbBuilder` with the specified path and object store.

<ParamField path="path" type="P: Into<Path>" required>
  The path to the database in object storage
</ParamField>

<ParamField path="object_store" type="Arc<dyn ObjectStore>" required>
  The object store implementation for persistence
</ParamField>

<ResponseField name="DbBuilder" type="DbBuilder<P>">
  Returns a new builder instance
</ResponseField>

```rust
use slatedb::{Db, Error};
use slatedb::object_store::memory::InMemory;
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<(), Error> {
    let object_store = Arc::new(InMemory::new());
    let db = Db::builder("test_db", object_store)
        .build()
        .await?;
    Ok(())
}
```

## Configuration Methods

### with_settings

```rust
pub fn with_settings(mut self, settings: Settings) -> Self
```

Sets custom database settings.

<ParamField path="settings" type="Settings" required>
  Database settings including performance tuning parameters
</ParamField>

<ResponseField name="DbBuilder" type="DbBuilder<P>">
  Returns the builder for method chaining
</ResponseField>

```rust
use slatedb::config::Settings;

let db = Db::builder("test_db", object_store)
    .with_settings(Settings {
        min_filter_keys: 2000,
        ..Default::default()
    })
    .build()
    .await?;
```

### with_sst_block_size

```rust
pub fn with_sst_block_size(mut self, block_size: SstBlockSize) -> Self
```

Sets the block size for SSTable blocks.

<ParamField path="block_size" type="SstBlockSize" required>
  The block size (1KB, 2KB, 4KB, 8KB, 16KB, 32KB, or 64KB)
</ParamField>

<ResponseField name="DbBuilder" type="DbBuilder<P>">
  Returns the builder for method chaining
</ResponseField>

```rust
use slatedb::config::SstBlockSize;

let db = Db::builder("test_db", object_store)
    .with_sst_block_size(SstBlockSize::Block8Kib)
    .build()
    .await?;
```

<Note>
Blocks are the unit of reading and caching. Smaller blocks reduce read amplification but increase metadata overhead. Larger blocks are better for sequential scans but may waste bandwidth for point lookups.
</Note>

### with_merge_operator

```rust
pub fn with_merge_operator(mut self, merge_operator: MergeOperatorType) -> Self
```

Sets a merge operator for the database.

<ParamField path="merge_operator" type="Arc<dyn MergeOperator>" required>
  The merge operator implementation
</ParamField>

<ResponseField name="DbBuilder" type="DbBuilder<P>">
  Returns the builder for method chaining
</ResponseField>

```rust
use slatedb::{MergeOperator, MergeOperatorError};
use bytes::Bytes;
use std::sync::Arc;

struct StringConcatMergeOperator;

impl MergeOperator for StringConcatMergeOperator {
    fn merge(&self, _key: &Bytes, existing_value: Option<Bytes>, value: Bytes) 
        -> Result<Bytes, MergeOperatorError> {
        let mut result = existing_value.unwrap_or_default().as_ref().to_vec();
        result.extend_from_slice(&value);
        Ok(Bytes::from(result))
    }
}

let db = Db::builder("test_db", object_store)
    .with_merge_operator(Arc::new(StringConcatMergeOperator))
    .build()
    .await?;
```

### with_block_transformer

<ResponseField name="DbBuilder" type="DbBuilder<P>">

</ResponseField>

<Info>

## Building the Database

### build

```rust
pub async fn build(self) -> Result<Db, Error>
```

Builds and opens the database with the configured options.

<ResponseField name="Result" type="Result<Db, Error>">
  Returns a `Db` instance on success, or an error if the database could not be opened
</ResponseField>

```rust
let db = Db::builder("test_db", object_store)
    .with_settings(Settings::default())
    .with_sst_block_size(SstBlockSize::Block8Kib)
    .build()
    .await?;
```
