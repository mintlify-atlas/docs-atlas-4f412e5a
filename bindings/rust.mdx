The Rust API is the native interface to SlateDB. All other language bindings are built on top of the Rust core.

## Installation

Add SlateDB to your `Cargo.toml`:

```toml
[dependencies]
slatedb = "0.2"
```

### Feature Flags

SlateDB supports several optional features:

```toml
[dependencies]
slatedb = { version = "0.2", features = ["aws", "azure", "snappy"] }
```

- **`aws`** (default): AWS S3 support
- **`azure`**: Azure Blob Storage support
- **`snappy`**: Snappy compression
- **`zlib`**: Zlib compression
- **`lz4`**: LZ4 compression
- **`zstd`**: Zstandard compression
- **`foyer`** (default): Foyer block cache
- **`moka`**: Moka block cache
- **`compaction_filters`**: Enable compaction filters API

## Basic Usage

### Opening a Database

```rust
use slatedb::{Db, DbBuilder};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Simple open with defaults
    let db = Db::open(
        "/tmp/my-db",
        object_store::memory::InMemory::new(),
    ).await?;

    Ok(())
}
```

### Write Operations

```rust
use bytes::Bytes;

// Put a key-value pair
db.put(b"key", b"value").await;

// Get a value
let value = db.get(b"key").await?;
assert_eq!(value, Some(Bytes::from("value")));

// Delete a key
db.delete(b"key").await;
```

### Write Batches

Batch multiple operations atomically:

```rust
use slatedb::WriteBatch;

let mut batch = WriteBatch::new();
batch.put(b"key1", b"value1");
batch.put(b"key2", b"value2");
batch.delete(b"old_key");

db.write(batch).await?;
```

### Range Scans

Iterate over key ranges:

```rust
// Scan all keys with prefix "user:"
let mut iter = db.scan_prefix(b"user:").await?;
while let Some(result) = iter.next().await {
    let (key, value) = result?;
    println!("{}={}", String::from_utf8_lossy(&key), String::from_utf8_lossy(&value));
}
```

## Advanced Configuration

### Using DbBuilder

Customize database settings:

```rust
use slatedb::{DbBuilder, Settings, SstBlockSize};

let settings = Settings {
    flush_interval: Some(std::time::Duration::from_millis(100)),
    l0_sst_size_bytes: Some(128 * 1024 * 1024), // 128MB
    ..Default::default()
};

let db = DbBuilder::new(
    "/tmp/my-db",
    object_store::memory::InMemory::new(),
)
.with_settings(settings)
.with_sst_block_size(SstBlockSize::SstBlockSize16Kib)
.build()
.await?;
```

### Object Store Configuration

#### AWS S3

```rust
use object_store::aws::AmazonS3Builder;

let object_store = AmazonS3Builder::new()
    .with_bucket_name("my-bucket")
    .with_region("us-west-2")
    .build()?;

let db = Db::open("/tmp/my-db", object_store).await?;
```

#### Azure Blob Storage

```rust
use object_store::azure::MicrosoftAzureBuilder;

let object_store = MicrosoftAzureBuilder::new()
    .with_account("myaccount")
    .with_container_name("my-container")
    .build()?;

let db = Db::open("/tmp/my-db", object_store).await?;
```

## Read-Only Access

### DbReader

Open a read-only view of the database:

```rust
use slatedb::{DbReader, DbReaderBuilder};

let reader = DbReaderBuilder::new(
    "/tmp/my-db",
    object_store::memory::InMemory::new(),
)
.build()
.await?;

let value = reader.get(b"key").await?;
```

### Snapshots

Create point-in-time snapshots:

```rust
let snapshot = db.snapshot();

// Read from snapshot (isolated from further writes)
let value = snapshot.get(b"key").await?;
```

## Transactions

### Snapshot Isolation

```rust
let txn = db.begin(slatedb::IsolationLevel::SnapshotIsolation).await?;
txn.put(b"key", b"value");
let value = txn.get(b"key").await?;
txn.commit().await?;
```

### Serializable Snapshot Isolation

```rust
let txn = db.begin(slatedb::IsolationLevel::SerializableSnapshotIsolation).await?;
txn.put(b"key1", b"value1");
txn.put(b"key2", b"value2");
txn.commit().await?;
```

## Merge Operators

Define custom merge logic:

```rust
use slatedb::{MergeOperator, MergeOperatorError};

struct CounterMerge;

impl MergeOperator for CounterMerge {
    fn merge(&self, _key: &[u8], existing: Option<&[u8]>, operands: &[&[u8]]) -> Result<Vec<u8>, MergeOperatorError> {
        let mut count = existing
            .and_then(|b| std::str::from_utf8(b).ok())
            .and_then(|s| s.parse::<i64>().ok())
            .unwrap_or(0);
        
        for operand in operands {
            if let Ok(delta) = std::str::from_utf8(operand).and_then(|s| s.parse::<i64>()) {
                count += delta;
            }
        }
        
        Ok(count.to_string().into_bytes())
    }
}

let db = DbBuilder::new("/tmp/my-db", object_store)
    .with_merge_operator(Arc::new(CounterMerge))
    .build()
    .await?;

db.merge(b"counter", b"1").await;
db.merge(b"counter", b"5").await;
let value = db.get(b"counter").await?; // Returns "6"
```

## TTL (Time-To-Live)

Set expiration times for keys:

```rust
use std::time::Duration;

// Key expires after 1 hour
db.put_with_options(
    b"session:123",
    b"data",
    Some(Duration::from_secs(3600)),
    None,
).await;
```

## Compaction

### Manual Compaction

```rust
use slatedb::{CompactorBuilder, GarbageCollectorBuilder};

// Run compactor
let compactor = CompactorBuilder::new(object_store.clone())
    .build();
compactor.run().await?;

// Run garbage collector
let gc = GarbageCollectorBuilder::new(object_store.clone())
    .build();
gc.run().await?;
```

## Performance Considerations

### Block Caching

SlateDB uses block caches to reduce object storage reads:

```rust
use slatedb::Settings;

let settings = Settings {
    object_store_cache_options: Some(ObjectStoreCacheOptions {
        max_cache_size_bytes: 1024 * 1024 * 1024, // 1GB cache
        part_size_bytes: 4096,
        ..Default::default()
    }),
    ..Default::default()
};
```

### Compression

Enable compression to reduce storage costs:

```rust
let settings = Settings {
    compression_codec: Some("snappy".to_string()),
    ..Default::default()
};
```

### Write Buffering

Adjust flush intervals for write-heavy workloads:

```rust
let settings = Settings {
    flush_interval: Some(Duration::from_millis(100)),
    max_unflushed_bytes: Some(64 * 1024 * 1024), // 64MB
    ..Default::default()
};
```

## Error Handling

```rust
use slatedb::{Error, ErrorKind};

match db.get(b"key").await {
    Ok(Some(value)) => println!("Found: {:?}", value),
    Ok(None) => println!("Key not found"),
    Err(Error { kind: ErrorKind::Closed, .. }) => {
        println!("Database is closed");
    }
    Err(e) => println!("Error: {}", e),
}
```

## Thread Safety

`Db`, `DbReader`, and `DbSnapshot` are all `Send + Sync` and can be safely shared across threads:

```rust
use std::sync::Arc;

let db = Arc::new(db);
let db_clone = db.clone();

tokio::spawn(async move {
    db_clone.put(b"key", b"value").await.unwrap();
});
```
